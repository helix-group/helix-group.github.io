Deploy our system

Requirements:
  • Your faxes are received as PDFs
  • COVID Cases are reported via one or more standard forms
  • You have an IT technician who can deploy a Python script

Frequently Asked Questions

Answers to some common questions

  • Can I deploy Fast Fax if I have a different template form?

    The models that power Fast Fax need to be trained each time a new form is added. If Fast Fax doesn't already recognize your form, please fill our our Fast Fax Inquiry form here.

  • Does the system require a powerful computer?

    No, Fast Fax is currently deployed on a computer with 8GB of RAM and an Intel® Xeon® Processor E5-2690 v3.

  • Is it freely available?

    Yes, the current system is freely available for use under the MIT open source license.
